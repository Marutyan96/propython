Циклы в Python

Циклы в Python используются для повторения блока кода несколько раз. Это позволяет автоматизировать задачи и обрабатывать данные более эффективно. В этом уроке мы рассмотрим два основных типа циклов: for и while.

Цикл for

Цикл for используется для перебора элементов в последовательностях, таких как списки, строки или диапазоны.

Пример 1: Перебор списка

fruits = ["яблоко", "банан", "вишня"]

for fruit in fruits:
    print(fruit)
Пример 2: Использование функции range()

Функция range() позволяет создавать последовательности чисел.


for i in range(5):
    print(f"Итерация {i}")

Цикл while

Цикл while выполняет блок кода, пока условие истинно.

Пример 3: Использование цикла while

count = 0

while count < 5:
    print(f"Счетчик: {count}")
    count += 1

Прерывание и продолжение цикла

Вы можете использовать ключевые слова break и continue для управления выполнением циклов.

Пример 4: Использование break

for i in range(10):
    if i == 5:
        break
    print(i)
Пример 5: Использование continue

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Заключение

Циклы являются мощным инструментом в Python, позволяющим выполнять повторяющиеся задачи с минимальными усилиями. Попробуйте создать свои собственные циклы и поэкспериментировать с различными условиями!
