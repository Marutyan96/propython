Арифметические операции с числами в Python

В Python вы можете выполнять различные арифметические операции с числами. В этом уроке мы рассмотрим основные операции: сложение, вычитание, умножение, деление, целочисленное деление, остаток от деления и возведение в степень.

Шаг 1: Сложение

Сложение выполняется с помощью оператора +.


a = 5
b = 3
result = a + b
print(f"Сложение: {result}")  # Вывод: Сложение: 8

Шаг 2: Вычитание

Вычитание выполняется с помощью оператора -.


result = a - b
print(f"Вычитание: {result}")  # Вывод: Вычитание: 2

Шаг 3: Умножение

Умножение выполняется с помощью оператора *.


result = a * b
print(f"Умножение: {result}")  # Вывод: Умножение: 15

Шаг 4: Деление

Деление выполняется с помощью оператора /.


result = a / b
print(f"Деление: {result}")  # Вывод: Деление: 1.6667

Шаг 5: Целочисленное деление

Целочисленное деление выполняется с помощью оператора //.


result = a // b
print(f"Целочисленное деление: {result}")  # Вывод: Целочисленное деление: 1

Шаг 6: Остаток от деления

Остаток от деления выполняется с помощью оператора %.


result = a % b
print(f"Остаток от деления: {result}")  # Вывод: Остаток от деления: 2

Шаг 7: Возведение в степень

Возведение в степень выполняется с помощью оператора **.


result = a ** b
print(f"Возведение в степень: {result}")  # Вывод: Возведение в степень: 125

Шаг 8: Ввод данных от пользователя

Теперь давайте позволим пользователю вводить два числа и выполнять все операции над ними.


try:
    a = int(input("Введите первое число: "))
    b = int(input("Введите второе число: "))

    print(f"Сложение: {a + b}")
    print(f"Вычитание: {a - b}")
    print(f"Умножение: {a * b}")
    print(f"Деление: {a / b}")
    print(f"Целочисленное деление: {a // b}")
    print(f"Остаток от деления: {a % b}")
    print(f"Возведение в степень: {a ** b}")
        
except ValueError:
    print("Пожалуйста, введите корректные числа.")

Заключение
