Введение в написание программ на Python

Шаг 1: Создание нового проекта

1.1. Открытие VSCode

Запустите Visual Studio Code.

1.2. Создание нового файла

Создайте новый файл с расширением .py (например, first_program.py).

Шаг 2: Написание первой программы

2.1. Написание кода

Введите следующий код в созданном файле:

print("Добро пожаловать в программирование на Python!")
2.2. Сохранение файла

Сохраните файл, нажав Ctrl + S или выбрав File > Save из меню.

Шаг 3: Запуск программы

3.1. Открытие терминала

Откройте встроенный терминал в VSCode, нажав Ctrl + ` (обратная кавычка) или выбрав Terminal > New Terminal из меню.

3.2. Переход к папке с файлом

Убедитесь, что вы находитесь в папке, где сохранен ваш файл first_program.py. Используйте команду cd для перехода в нужную директорию:

cd путь/к/вашей/папке
3.3. Запуск программы

Введите следующую команду для запуска вашей программы:

python first_program.py

или

python3 first_program.py
3.4. Ожидаемый результат

Вы должны увидеть вывод:

Добро пожаловать в программирование на Python!

Заключение

Поздравляем, вы только что написали и запустили свою первую программу на Python! Продолжайте изучать язык, экспериментируйте с кодом и открывайте для себя новые возможности программирования.
