Установка и настройка Visual Studio Code для Python

Шаг 1: Установка Visual Studio Code

1.1. Загрузка VSCode

Перейдите на официальный сайт Visual Studio Code: code.visualstudio.com.

На главной странице нажмите на кнопку для загрузки последней версии для вашей операционной системы.

1.2. Установка VSCode

Для Windows:

  • Запустите загруженный установочный файл.
  • Следуйте инструкциям установщика.

Для macOS:

  • Откройте загруженный файл .dmg и перетащите VSCode в папку Applications.

Для Linux:

  • Откройте терминал и введите следующую команду:
sudo snap install code --classic

Шаг 2: Установка расширений для Python

После установки откройте VSCode и выполните следующие действия:

2.1. Открытие панели расширений

Нажмите на иконку расширений на боковой панели или используйте сочетание клавиш Ctrl + Shift + X.

2.2. Установка расширения Python

В строке поиска введите Python и установите расширение от Microsoft.

2.3. Установка расширения Pylance (опционально)

Также вы можете установить расширение Pylance для улучшенной поддержки IntelliSense.

Шаг 3: Настройка интерпретатора Python

После установки расширений необходимо выбрать интерпретатор Python:

3.1. Открытие командной палитры

Нажмите Ctrl + Shift + P и введите Python: Select Interpreter

3.2. Выбор интерпретатора

Выберите установленный интерпретатор Python из списка.

Шаг 4: Создание и запуск первой программы

4.1. Создание нового файла

Создайте новый файл с расширением .py (например, hello.py) в VSCode.

4.2. Написание кода

Введите следующий код:

print("Hello, World!")
4.3. Запуск программы

Запустите программу, нажав F5 или выбрав Run > Start Debugging из меню.

Вы должны увидеть вывод:

Hello, World!
