Кортежи в Python

Кортежи в Python — это упорядоченные, неизменяемые коллекции, которые могут содержать элементы различных типов. Они часто используются для хранения связанных данных и являются полезным инструментом в программировании.

Создание кортежа

Кортежи создаются с помощью круглых скобок (). Вот пример создания простого кортежа:

        
my_tuple = (1, 2, 3)
print(my_tuple)  # Вывод: (1, 2, 3)

Кортежи с разными типами данных

Кортежи могут содержать элементы различных типов, включая числа, строки и даже другие кортежи. Пример:

        
mixed_tuple = (1, "два", 3.0, (4, 5))
print(mixed_tuple)  # Вывод: (1, 'два', 3.0, (4, 5))

Доступ к элементам кортежа

Элементы кортежа можно получить по индексу, начиная с 0:

        
print(my_tuple[0])  # Вывод: 1
print(mixed_tuple[1])  # Вывод: 'два'

Итерация по кортежу

Итерация по элементам кортежа может быть выполнена с помощью цикла for. Вот пример:


for item in my_tuple:
    print(item)

Кортежи и их длина

Вы можете узнать количество элементов в кортеже с помощью функции len():

        
length = len(my_tuple)
print(f"Количество элементов: {length}")  # Вывод: Количество элементов: 3

Объединение кортежей

Кортежи можно объединять с помощью оператора +. Например:

        
another_tuple = (4, 5)
combined_tuple = my_tuple + another_tuple
print(combined_tuple)  # Вывод: (1, 2, 3, 4, 5)

Неизменяемость кортежей

Кортежи являются неизменяемыми, поэтому вы не можете изменять их элементы после создания. Попытка сделать это приведет к ошибке:

        
# Это вызовет ошибку:
my_tuple[0] = 10  # TypeError: 'tuple' object does not support item assignment

Заключение

Кортежи являются полезным инструментом для хранения данных, когда вам не нужно изменять содержимое. Попробуйте создать свои собственные кортежи и поэкспериментировать с их использованием! Также помните, что кортежи могут быть вложенными, что позволяет создавать более сложные структуры данных.
