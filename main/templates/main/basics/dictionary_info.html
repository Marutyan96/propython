Урок по словарям в Python

Словари в Python представляют собой неупорядоченные коллекции пар "ключ-значение". Они используются для хранения данных, которые можно быстро извлекать по уникальному ключу.

Создание словаря

Словари создаются с помощью фигурных скобок {} или с помощью функции dict().


my_dict = {}"name": "Алексей", "age": 30
print(my_dict)  # Вывод: {'name': 'Алексей', 'age': 30}

Доступ к значениям словаря

Доступ к значениям осуществляется с помощью ключа.

            
print(my_dict["name"])  # Вывод: Алексей

Изменение значений в словаре

Вы можете изменять значения, используя ключ.

            
my_dict["age"] = 31
print(my_dict)  # Вывод: {'name': 'Алексей', 'age': 31}

Добавление новых пар "ключ-значение"

Новые пары можно добавлять, используя новый ключ.

            
my_dict["city"] = "Москва"
print(my_dict)  # Вывод: {'name': 'Алексей', 'age': 31, 'city': 'Москва'}

Удаление пар "ключ-значение"

Вы можете удалять пары с помощью оператора del.


del my_dict["age"]
print(my_dict)  # Вывод: {'name': 'Алексей', 'city': 'Москва'}

Методы словарей

Словари имеют несколько полезных методов, таких как keys(), values() и items().


print(my_dict.keys())   # Вывод: dict_keys(['name', 'city'])
print(my_dict.values()) # Вывод: dict_values(['Алексей', 'Москва'])
print(my_dict.items())  # Вывод: dict_items([('name', 'Алексей'), ('city', 'Москва')])

Проверка наличия ключа

Вы можете проверять наличие ключа в словаре с помощью оператора in.


if "name" in my_dict:
    print("Ключ 'name' существует.")

Заключение

Словари являются мощным инструментом для хранения и управления данными в Python. Попробуйте создать свои собственные словари и поэкспериментировать с их методами!
