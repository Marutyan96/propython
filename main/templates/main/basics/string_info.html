Работа со строками в Python

Строки в Python представляют собой последовательности символов. Они являются одним из основных типов данных и предоставляют множество методов для обработки текстовой информации.

Создание строк

Строки могут быть созданы с помощью одинарных ' ' или двойных " " кавычек.


string1 = 'Привет'
string2 = "Мир"
print(string1 + " " + string2)  # Вывод: Привет Мир

Доступ к символам строки

Вы можете получить доступ к отдельным символам строки, используя индексы.


my_string = "Python"
print(my_string[0])  # Вывод: P
print(my_string[-1])  # Вывод: n

Срезы строк

Срезы позволяют извлекать подстроки из строки.


my_string = "Python"
substring = my_string[1:4]
print(substring)  # Вывод: yth

Методы строк

Строки имеют множество полезных методов. Вот несколько из них:

1. lower() и upper()

Методы для изменения регистра строки.


my_string = "Python"
print(my_string.lower())  # Вывод: python
print(my_string.upper())  # Вывод: PYTHON
2. strip()

Метод для удаления пробелов в начале и конце строки.


my_string_with_spaces = "  Python  "
print(my_string_with_spaces.strip())  # Вывод: Python
3. replace()

Метод для замены подстроки в строке.


my_string = "Python"
new_string = my_string.replace("Python", "Java")
print(new_string)  # Вывод: Java
4. split()

Метод для разделения строки на список подстрок.


sentence = "Привет, мир!"
words = sentence.split(", ")
print(words)  # Вывод: ['Привет', 'мир!']

Конкатенация строк

Строки можно объединять с помощью оператора +.


greeting = "Привет"
name = "Алексей"
message = greeting + ", " + name
print(message)  # Вывод: Привет, Алексей
