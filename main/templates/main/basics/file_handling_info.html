Работа с файлами в Python

В Python работа с файлами является важной частью программирования. Вы можете открывать, читать, записывать и закрывать файлы. В этом уроке мы рассмотрим основные операции с файлами.

Открытие файла

Чтобы открыть файл, используйте функцию open(). Она принимает два аргумента: имя файла и режим открытия.


file = open('example.txt', 'r')  # 'r' - режим чтения

Чтение из файла

После открытия файла вы можете читать его содержимое с помощью различных методов:

  • read() - читает весь файл.
  • readline() - читает одну строку.
  • readlines() - читает все строки и возвращает их в виде списка.
Пример чтения файла

content = file.read()
print(content)

Закрытие файла

После завершения работы с файлом его необходимо закрыть, чтобы освободить ресурсы:


file.close()

Запись в файл

Чтобы записать данные в файл, откройте его в режиме записи 'w' или добавления 'a'.

Пример записи в файл

with open('output.txt', 'w') as file:
    file.write('Привет, мир!')

Использование конструкции with

Рекомендуется использовать конструкцию with для работы с файлами, так как она автоматически закрывает файл после выхода из блока:


with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

Заключение

Работа с файлами в Python позволяет вам сохранять и загружать данные. Используйте правильные режимы открытия и конструкцию with для эффективной работы с файлами.
