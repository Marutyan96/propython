Функция range в Python

Функция range() в Python используется для генерации последовательности чисел. Она часто применяется в циклах for для итерации по числовым диапазонам.

Основные понятия функции range

  • range(stop): Генерирует последовательность чисел от 0 до stop (не включая stop).
  • range(start, stop): Генерирует последовательность чисел от start до stop (не включая stop).
  • range(start, stop, step): Генерирует последовательность чисел от start до stop с шагом step.

Шаг 1: Использование range без параметров

Давайте начнем с простого примера использования range() без параметров.


for i in range(5):
    print(i)

Шаг 2: Использование range с двумя параметрами

Теперь рассмотрим использование range() с двумя параметрами: start и stop.


for i in range(2, 8):
    print(i)

Шаг 3: Использование range с тремя параметрами

Теперь добавим третий параметр для задания шага.


for i in range(1, 10, 2):
    print(i)

Шаг 4: Преобразование range в список

Вы можете преобразовать объект range в список, чтобы увидеть все значения в нем.


my_range = list(range(5))
print(my_range)  # Вывод: [0, 1, 2, 3, 4]

Заключение

Функция range() является мощным инструментом для работы с последовательностями чисел в Python. Она широко используется в циклах и позволяет легко генерировать числовые последовательности. Попробуйте использовать range() в своих проектах и экспериментах!
