Лямбда-функции в Python

Лямбда-функции — это анонимные функции в Python, которые могут иметь любое количество аргументов, но только одно выражение. Они часто используются для создания небольших функций на лету.

Создание лямбда-функции

Лямбда-функцию можно создать с помощью ключевого слова lambda. Вот пример: 

    
add = lambda x, y: x + y
print(add(2, 3))  # Вывод: 5

Лямбда-функции с несколькими аргументами

Лямбда-функции могут принимать несколько аргументов. Например:


multiply = lambda a, b: a * b
print(multiply(4, 5))  # Вывод: 20

Использование лямбда-функций с функциями высшего порядка

Лямбда-функции часто используются в сочетании с функциями высшего порядка, такими как map(), filter() и reduce(). Например:

Пример с map()
    
numbers = [1, 2, 3, 4]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # Вывод: [1, 4, 9, 16]
Пример с filter()

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # Вывод: [2, 4]

Лямбда-функции и сортировка

Лямбда-функции также могут использоваться для сортировки списков. Например: 

    
data = [(1, 'один'), (3, 'три'), (2, 'два')]
sorted_data = sorted(data, key=lambda x: x[0])
print(sorted_data)  # Вывод: [(1, 'один'), (2, 'два'), (3, 'три')]

Заключение

Лямбда-функции — это мощный инструмент для создания небольших, анонимных функций в Python. Они особенно полезны в сочетании с функциями высшего порядка. Попробуйте использовать лямбда-функции в своих проектах и оцените их удобство!
