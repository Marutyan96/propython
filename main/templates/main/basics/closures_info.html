Замыкания в Python

Замыкание — это функция, которая запоминает значение переменных из своей внешней области видимости даже после того, как эта внешняя функция завершила выполнение. Это позволяет создавать функции с сохранением состояния.

Как работают замыкания?

Когда внутренняя функция ссылается на переменные внешней функции, создается замыкание. Важно, что замыкания позволяют сохранять состояние между вызовами функции.

Пример замыкания


def outer_function(x):
    def inner_function(y):
        return x + y # доступ к переменной x из внешней функции
    
    return inner_function # возвращаем внутреннюю функцию

closure = outer_function(10) # создаем замыкание с x = 10
print(closure(5)) # выводит 15 (10 + 5)

Применение замыканий

Замыкания часто используются для создания функций с состоянием, например, в обработчиках событий или для создания фабрик функций.

Пример: Фабрика функций

def make_multiplier(factor):
    def multiplier(x):
        return x * factor # умножаем x на factor
    
    return multiplier # возвращаем функцию умножения

double = make_multiplier(2) # создаем функцию для удвоения
triple = make_multiplier(3) # создаем функцию для утроения

print(double(5)) # выводит 10
print(triple(5)) # выводит 15

Итоги

Замыкания в Python позволяют создавать функции, которые могут запоминать значения переменных из внешней области видимости. Это мощный инструмент для управления состоянием и создания функций с определенной логикой.
