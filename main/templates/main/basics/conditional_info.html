Условные выражения в Python
Операции сравнения в Python

Ряд операций представляют условные выражения. Все эти операции принимают два операнда и возвращают логическое значение, которое в Python представляет тип bool. Существует только два логических значения - True (выражение истинно) и False (выражение ложно).

Операции сравнения

Простейшие условные выражения представляют операции сравнения, которые сравнивают два значения. Python поддерживает следующие операции сравнения:

  • ==: Возвращает True, если оба операнда равны. Иначе возвращает False.
  • !=: Возвращает True, если оба операнда НЕ равны. Иначе возвращает False.
  • >: (больше чем): Возвращает True, если первый операнд больше второго.
  • <: (меньше чем): Возвращает True, если первый операнд меньше второго.
  • >=: (больше или равно): Возвращает True, если первый операнд больше или равен второму.
  • <=: (меньше или равно): Возвращает True, если первый операнд меньше или равен второму.

Примеры операций сравнения:


a = 5
b = 6
result = 5 == 6  # сохраняем результат операции в переменную
print(result)  # False - 5 не равно 6
print(a != b)  # True
print(a > b)  # False - 5 меньше 6
print(a < b)  # True

bool1 = True
bool2 = False
print(bool1 == bool2)  # False - bool1 не равно bool2

Операции сравнения могут сравнивать различные объекты - строки, числа, логические значения, однако оба операнда операции должны представлять один и тот же тип.

Условные выражения позволяют выполнять различные действия в зависимости от выполнения определенных условий. Это важный аспект программирования, который позволяет создавать динамические и интерактивные программы.

Логические операции в Python

Логические операции используются для комбинирования или инверсии логических значений. В Python существуют следующие логические операторы:

  • and: Возвращает True, если оба операнда истинны.
  • or: Возвращает True, если хотя бы один из операндов истинен.
  • not: Инвертирует логическое значение (возвращает True, если операнд ложен, и наоборот).
  • in: Проверяет, содержится ли элемент в последовательности (например, в списке или строке).
  • not in: Проверяет, не содержится ли элемент в последовательности.

Примеры логических операций:


x = 10
y = 20
z = 30

print(x < y and y < z)  # True, так как 10 < 20 и 20 < 30
print(x > y or y < z)  # True, так как 20 < 30
print(not (x > y))  # True, так как x не больше y

my_list = ['apple', 'banana', 'cherry']
print('banana' in my_list)  # True, 'banana' в списке
print('grape' not in my_list)  # True, 'grape' не в списке

Логические операции часто используются в условных выражениях, чтобы контролировать выполнение кода в зависимости от истинности выражений.

Условия : if, else, elif

В Python конструкции if, else и elif используются для выполнения различных блоков кода в зависимости от условий.

Синтаксис


if условие:
    # код, выполняемый если условие истинно
    ... 
elif другое_условие:
    # код, выполняемый если другое условие истинно
    ... 
else:
    # код, выполняемый если ни одно условие не истинно
    ...
Пример 1: Проверка четности числа

number = 7

if number % 2 == 0:
    print('Число четное')
else:
    print('Число нечетное')
Пример 2: Оценка по баллам

score = 85

if score >= 90:
    print('Оценка: A')
elif score >= 80:
    print('Оценка: B')
elif score >= 70:
    print('Оценка: C')
else:
    print('Оценка: D')
Пример 3: Определение времени суток

hour = 14

if hour >= 5 and hour < 12:
    print('Доброе утро!')
elif hour >= 12 and hour < 17:
    print('Добрый день!')
else:
    print('Добрый вечер!')

Итоги

Конструкции if, elif и else позволяют управлять потоком выполнения программы, выполняя разные действия в зависимости от условий. Это позволяет создавать более сложные логические структуры в вашем коде.
