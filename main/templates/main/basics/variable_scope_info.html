Область видимости переменных в Python

Область видимости переменных определяет, где в коде можно использовать переменные. В Python существует несколько уровней области видимости, включая локальную, глобальную и встроенную области видимости.

Типы области видимости

  • Локальная область видимости: Переменные, определенные внутри функции, доступны только в этой функции.
  • Глобальная область видимости: Переменные, определенные вне всех функций, доступны в любой части кода.
  • Встроенная область видимости: Переменные и функции, доступные по умолчанию в Python, такие как print() и len().

Примеры области видимости переменных

Пример 1: Локальная область видимости

def my_function():
    local_var = 'Я локальная переменная' # локальная переменная
    print(local_var) # выводим локальную переменную

my_function() # вызываем функцию
print('Локальная переменная доступна здесь: ', local_var) # вызовет ошибку, так как local_var не доступна здесь
Пример 2: Глобальная область видимости

global_var = 'Я глобальная переменная' # глобальная переменная

def my_function():
    print('Глобальная переменная внутри функции: ', global_var) # доступ к глобальной переменной

my_function() # вызываем функцию
print('Глобальная переменная снаружи: ', global_var) # доступ к глобальной переменной
Пример 3: Изменение глобальной переменной внутри функции

global_var = 10 # глобальная переменная

def modify_global():
    global global_var # объявляем, что будем использовать глобальную переменную
    global_var += 5 # изменяем глобальную переменную

modify_global() # вызываем функцию
print('Измененная глобальная переменная: ', global_var) # выводим измененную глобальную переменную
Пример 4: Встроенные переменные

numbers = [1, 2, 3]
print('Количество элементов в списке: ', len(numbers)) # используем встроенную функцию len()

Итоги

Область видимости переменных в Python играет важную роль в управлении доступом к переменным в различных частях программы. Понимание локальной и глобальной области видимости поможет вам писать более понятный и безопасный код.
