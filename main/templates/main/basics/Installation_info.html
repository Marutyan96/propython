Установка Python и написание первой программы

Шаг 1: Установка Python

1.1. Загрузка Python

Перейдите на официальный сайт Python: python.org.

На главной странице вы увидите кнопку для загрузки последней версии Python. Нажмите на нее.

1.2. Установка Python

Для Windows:

  • Запустите загруженный установочный файл.
  • Обязательно отметьте галочку "Add Python to PATH" перед нажатием на "Install Now".

Для macOS:

  • Откройте загруженный файл .pkg и следуйте инструкциям по установке.

Для Linux:

  • Откройте терминал и введите следующую команду:
sudo apt update

sudo apt install python3

Шаг 2: Проверка установки Python

После установки откройте командную строку (или терминал) и введите:

python --version

или

python3 --version

Вы должны увидеть установленную версию Python.

Шаг 3: Написание первой программы

3.1. Открытие текстового редактора

Вы можете использовать любой текстовый редактор, например:

  • Notepad (Windows)
  • TextEdit (macOS)
  • nano или vim (Linux)
3.2. Написание кода

Создайте новый файл и введите следующий код:

print("Hello, World!")
3.3. Сохранение файла

Сохраните файл с именем hello.py. Убедитесь, что файл имеет расширение .py.

Шаг 4: Запуск программы

4.1. Открытие терминала или командной строки
  • Windows: Нажмите Win + R, введите cmd и нажмите Enter.
  • macOS: Откройте Terminal из Applications > Utilities.
  • Linux: Откройте терминал.
4.2. Переход к папке с файлом

Используйте команду cd, чтобы перейти в папку, где вы сохранили файл hello.py. Например:

cd путь/к/вашей/папке
4.3. Запуск программы

Введите следующую команду:

python hello.py

или

python3 hello.py

Вы должны увидеть вывод:

Hello, World!
