Множества в Python

Множества в Python представляют собой неупорядоченные коллекции уникальных элементов. Они используются для хранения данных, когда важна только уникальность значений.

Создание множества

Множества создаются с помощью фигурных скобок {} или с помощью функции set().


my_set = {1, 2, 3, 4}
print(my_set)  # Вывод: {1, 2, 3, 4}

Создание пустого множества

Чтобы создать пустое множество, используйте функцию set(), так как фигурные скобки создают пустой словарь.


empty_set = set()
print(empty_set)  # Вывод: set()

Добавление элементов в множество

Элементы добавляются с помощью метода add().


my_set.add(5)
print(my_set)  # Вывод: {1, 2, 3, 4, 5}

Удаление элементов из множества

Элементы можно удалять с помощью метода remove() или discard().


my_set.remove(2)
print(my_set)  # Вывод: {1, 3, 4, 5}

my_set.discard(10)  # Не вызывает ошибку, если элемента нет

Проверка наличия элемента

Вы можете проверять наличие элемента в множестве с помощью оператора in.


if 3 in my_set:
    print("3 есть в множестве.")

Объединение и пересечение множеств

Вы можете объединять множества с помощью метода union() или оператора |, а также находить пересечение с помощью метода intersection() или оператора &.


set_a = {1, 2, 3}
set_b = {3, 4, 5}

union_set = set_a | set_b
print(union_set)  # Вывод: {1, 2, 3, 4, 5}

intersection_set = set_a & set_b
print(intersection_set)  # Вывод: {3}

Заключение

Множества являются мощным инструментом для работы с уникальными данными в Python. Попробуйте создать свои собственные множества и поэкспериментировать с их методами!
