Переменные и типы данных в Python

Переменные в Python используются для хранения данных. Они могут принимать различные типы данных, такие как числа, строки, списки и другие. Понимание типов данных и работы с переменными является основой программирования на Python.

Основные понятия переменных

  • Переменная: Именованная область памяти, в которой хранится значение.
  • Присваивание: Процесс установки значения переменной с помощью оператора =.

Типы данных в Python

  • int: Целые числа.
  • float: Числа с плавающей запятой (десятичные числа).
  • str: Строки (последовательности символов).
  • bool: Логические значения True и False.
  • list: Списки (упорядоченные коллекции элементов).
  • dict: Словари (неупорядоченные коллекции пар ключ-значение).

Шаг 1: Присваивание переменных

Давайте начнем с простого примера присваивания переменных.


name = "Алексей"
age = 30
height = 1.75
is_student = False

Шаг 2: Вывод значений переменных

Теперь выведем значения переменных на экран.


print(name)       # Вывод: Алексей
print(age)        # Вывод: 30
print(height)     # Вывод: 1.75
print(is_student) # Вывод: False

Шаг 3: Изменение значений переменных

Переменные могут изменять свои значения в процессе выполнения программы.


age = age + 1
print(age) # Вывод: 31

Шаг 4: Типы данных

print(type(name))       # Вывод:

Шаг 5: Списки и словари

Рассмотрим, как использовать списки и словари в Python.


fruits = ["яблоко", "банан", "вишня"]
person = {"name": "Алексей", "age": 30}

print(fruits)   # Вывод: ['яблоко', 'банан', 'вишня']
print(person)   # Вывод: {'name': 'Алексей', 'age': 30}

Заключение

Переменные и типы данных являются основой программирования на Python. Понимание различных типов данных и умение работать с переменными поможет вам создавать более сложные программы. Попробуйте создавать свои собственные переменные и экспериментировать с различными типами данных!
