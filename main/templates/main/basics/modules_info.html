Определение и подключение модулей в Python

Модули в Python — это файлы, содержащие код, который можно повторно использовать в других программах. Они позволяют организовать код и делить его на логические части.

Создание модуля

Для создания модуля просто создайте файл с расширением .py. Например, создадим файл mymodule.py со следующим содержимым:


def greet(name):
    return f"Привет, {name}!"

Подключение модуля

Чтобы использовать функции из модуля, его нужно импортировать. Это можно сделать с помощью ключевого слова import.


import mymodule

message = mymodule.greet("Алексей")
print(message)  # Вывод: Привет, Алексей!

Импортирование отдельных функций

Если вы хотите импортировать только определённую функцию из модуля, используйте следующий синтаксис:


from mymodule import greet

message = greet("Мария")
print(message)  # Вывод: Привет, Мария!

Импортирование всех функций

Вы также можете импортировать все функции из модуля, используя *:


from mymodule import *

message = greet("Иван")
print(message)  # Вывод: Привет, Иван!

Создание пакетов

Пакеты — это специальные модули, которые могут содержать другие модули. Чтобы создать пакет, создайте папку и добавьте в неё файл __init__.py.

Например, структура пакета может выглядеть так:


    my_package/
        __init__.py
        mymodule.py

Импортирование из пакета

Чтобы импортировать модуль из пакета, используйте следующий синтаксис:


from my_package.mymodule import greet

message = greet("Ольга")
print(message)  # Вывод: Привет, Ольга!

Заключение

Модули в Python позволяют организовать код и повторно использовать его в различных проектах. Используйте модули и пакеты для улучшения структуры вашего кода и упрощения его поддержки.
