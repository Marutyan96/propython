Списки в Python

Списки в Python — это упорядоченные изменяемые коллекции, которые могут содержать элементы различных типов. Списки позволяют хранить и обрабатывать данные более эффективно. В этом уроке мы рассмотрим основные операции со списками.

Создание списка

Списки создаются с помощью квадратных скобок [].


fruits = ["яблоко", "банан", "вишня"]
print(fruits)  # Вывод: ['яблоко', 'банан', 'вишня']

Добавление элементов в список

Для добавления элементов в список используется метод append().


fruits.append("апельсин")
print(fruits)  # Вывод: ['яблоко', 'банан', 'вишня', 'апельсин']

Удаление элементов из списка

Для удаления элементов из списка используется метод remove() или оператор del.

      
fruits.remove("банан")  # Удаляем банан
print(fruits)  # Вывод: ['яблоко', 'вишня', 'апельсин']

del fruits[0]  # Удаляем первый элемент
print(fruits)  # Вывод: ['вишня', 'апельсин']

Итерация по списку

Итерация по элементам списка может быть выполнена с помощью цикла for.

      
for fruit in fruits:
  print(fruit)

Доступ к элементам списка

Элементы списка можно получить по индексу, начиная с 0.

print(fruits[1]) # Вывод: апельсин

Изменение элементов списка

Элементы списка можно изменять, обращаясь к ним по индексу.

      
fruits = ["яблоко", "банан", "вишня"]
fruits[1] = "груша"
print(fruits)  # Вывод: ['яблоко', 'груша', 'вишня']

Список и его длина

Вы можете узнать количество элементов в списке с помощью функции len().


fruits = ["яблоко", "банан", "вишня"]
length = len(fruits)
print(f"Количество фруктов: {length}")  # Вывод: Количество фруктов: 3

Заключение

Списки являются мощным инструментом для хранения и обработки данных в Python. Попробуйте создать свои собственные списки и поэкспериментировать с различными методами!
