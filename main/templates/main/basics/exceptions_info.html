Конструкция try...except в Python

Конструкция try...except в Python используется для обработки исключений, что позволяет вашему коду продолжать выполнение даже в случае возникновения ошибок. Это полезно для предотвращения аварийного завершения программы.

Основы конструкции try...except

Синтаксис конструкции выглядит следующим образом:


try:
    # Код, который может вызвать исключение
    result = 10 / 0  # Деление на ноль
except ZeroDivisionError:
    print('Ошибка: Деление на ноль!')

Обработка нескольких исключений

Вы можете обрабатывать несколько типов исключений, указав их через запятую:


try:
    value = int(input('Введите число: '))
    result = 10 / value
except (ValueError, ZeroDivisionError):
    print('Ошибка: Введите корректное число, отличное от нуля.')

Использование блока else

Вы также можете добавить блок else, который будет выполнен, если в блоке try не возникло исключений:


try:
    value = int(input('Введите число: '))
    result = 10 / value
except ZeroDivisionError:
    print('Ошибка: Деление на ноль!')
else:
    print('Результат: ' + result)

Использование блока finally

Блок finally выполняется в любом случае, независимо от того, произошло исключение или нет. Это полезно для выполнения завершающих операций, таких как закрытие файлов:


try:
    file = open('example.txt', 'r')
    content = file.read()
except FileNotFoundError:
    print('Файл не найден.')
finally:
    print('Завершение работы.')
    file.close()  # Закрытие файла

Заключение

Конструкция try...except позволяет вам обрабатывать ошибки и исключения, улучшая стабильность и надежность вашего кода. Используйте её для управления потенциальными проблемами в вашем приложении.
