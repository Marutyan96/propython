Консольный ввод и вывод в Python

В Python мы можем взаимодействовать с пользователем через консоль, получая данные с помощью ввода и отображая результаты с помощью вывода. В этом уроке мы рассмотрим, как использовать функции input() и print().

Шаг 1: Вывод данных на экран

Для вывода данных на экран используется функция print().

print("Привет, мир!") # Вывод: Привет, мир!

Шаг 2: Ввод данных от пользователя

Чтобы получить данные от пользователя, мы используем функцию input(). Она позволяет пользователю ввести данные с клавиатуры.


name = input("Введите ваше имя: ")
print(f"Привет, {name}!")  # Вывод: Привет, [имя пользователя]!

Шаг 3: Преобразование типов данных

Данные, полученные с помощью input(), всегда имеют тип str. Если вам нужно работать с числами, необходимо преобразовать их в нужный тип.


age = int(input("Введите ваш возраст: "))
print(f"Вам {age} лет!")  # Вывод: Вам [возраст] лет!

Шаг 4: Обработка ошибок ввода

Важно обрабатывать возможные ошибки ввода. Можно использовать блок try-except для обработки исключений.


try:
    age = int(input("Введите ваш возраст: "))
    print(f"Вам {age} лет!")
except ValueError:
    print("Пожалуйста, введите корректное число.")

Шаг 5: Множественный ввод данных

Можно также получить несколько данных от пользователя за один раз.


name, age = input("Введите ваше имя и возраст через пробел: ").split()
age = int(age)
print(f"Привет, {name}! Вам {age} лет!")

Заключение

Консольный ввод и вывод являются важными аспектами взаимодействия с пользователем в Python. Используя функции input() и print(), вы можете получать данные от пользователя и отображать результаты. Попробуйте создать свои собственные программы, используя консольный ввод и вывод!
