Преобразование типов в Python

Преобразование типов (или приведение типов) в Python позволяет изменять тип данных переменной. Это может быть полезно, когда необходимо выполнять операции между переменными разных типов.

Основные функции преобразования типов
  • int() — преобразует значение в целое число.
  • float() — преобразует значение в число с плавающей точкой.
  • str() — преобразует значение в строку.
  • list() — преобразует значение в список.
  • tuple() — преобразует значение в кортеж.
  • set() — преобразует значение в множество.
Примеры преобразования типов
Пример 1: Преобразование в целое число

num_str = '42' # строка, представляющая число
num_int = int(num_str) # преобразуем строку в целое число
print('Целое число: ', num_int) # выводим результат
Пример 2: Преобразование в число с плавающей точкой

float_str = '3.14' # строка, представляющая число с плавающей точкой
num_float = float(float_str) # преобразуем строку в число с плавающей точкой
print('Число с плавающей точкой: ', num_float) # выводим результат
Пример 3: Преобразование в строку

num = 100 # целое число
num_str = str(num) # преобразуем целое число в строку
print('Строка: ', num_str) # выводим результат
Пример 4: Преобразование списка в кортеж

my_list = [1, 2, 3] # список чисел
my_tuple = tuple(my_list) # преобразуем список в кортеж
print('Кортеж: ', my_tuple) # выводим результат
Пример 5: Преобразование кортежа в множество

my_tuple = (1, 2, 2) # кортеж с дублирующимися элементами
my_set = set(my_tuple) # преобразуем кортеж в множество (дубликаты удаляются)
print('Множество: ', my_set) # выводим результат
Итоги

Преобразование типов в Python — это мощный инструмент, который позволяет работать с данными различных типов. Используя функции преобразования, вы можете легко изменять типы переменных в зависимости от ваших нужд.
