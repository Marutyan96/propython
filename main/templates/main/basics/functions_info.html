Функции в Python

Функции в Python позволяют организовать код в логические блоки, которые можно вызывать несколько раз. Это помогает избежать дублирования кода и делает его более читаемым и управляемым. В этом уроке мы рассмотрим, как создавать и использовать функции в Python.

Определение функции

Функции определяются с помощью ключевого слова def, за которым следует имя функции и круглые скобки.


def greet(name):
    print(f"Привет, {name}!")

Вызов функции

Чтобы вызвать функцию, просто используйте её имя и передайте необходимые аргументы.

greet("Алексей") # Вывод: Привет, Алексей!

Функции с возвращаемыми значениями

Функции могут возвращать значения с помощью ключевого слова return.


def add(a, b):
    return a + b

result = add(5, 3)
print(f"Результат: {result}")  # Вывод: Результат: 8

Аргументы по умолчанию

Вы можете задавать значения по умолчанию для аргументов функции.


def greet(name, greeting = "Привет"):
    print(f"{greeting}, {name}!")

greet("Алексей")  # Вывод: Привет, Алексей!
greet("Мария", "Здравствуйте")  # Вывод: Здравствуйте, Мария!

Неопределенное количество аргументов

Вы можете использовать *args и **kwargs для передачи переменного количества аргументов.


def print_args(*args):
    for arg in args:
        print(arg)

print_args("Первый", "Второй", "Третий")
