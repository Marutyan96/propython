Основы FastAPI

FastAPI - это современный, быстрый (высокая производительность) веб-фреймворк для создания API на Python. Он основан на стандартных типах Python и позволяет быстро разрабатывать приложения с минимальными усилиями.

Установка FastAPI и Uvicorn

Чтобы установить FastAPI и Uvicorn (ASGI-сервер), используйте pip:

pip install fastapi uvicorn

Создание простого приложения

Давайте создадим простое веб-приложение, которое будет возвращать приветственное сообщение.

Пример кода

from fastapi import FastAPI

app = FastAPI()

@app.get('/')
def read_root():
    return {'message': 'Привет, мир!'}

Запуск приложения

Сохраните код в файл, например app.py, и запустите его с помощью Uvicorn:

uvicorn app:app --reload

После этого откройте браузер и перейдите по адресу http://127.0.0.1:8000/, чтобы увидеть сообщение "Привет, мир!".

Добавление маршрутов

Вы можете добавлять дополнительные маршруты для обработки различных URL. Например:


@app.get('/about')
def about():
    return {'message': 'Это страница о нас.'}

Использование параметров

FastAPI позволяет легко обрабатывать параметры в URL. Например, давайте добавим маршрут с параметром:


@app.get('/items/{item_id}')
def read_item(item_id: int):
    return {'item_id': item_id}

Заключение

FastAPI - это мощный инструмент для создания веб-приложений и API. В этом уроке мы рассмотрели основы FastAPI, установку, создание простого приложения, добавление маршрутов и использование параметров. Теперь вы можете начать разрабатывать свои собственные приложения на FastAPI!
