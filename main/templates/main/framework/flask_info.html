Основы Flask

Flask - это легковесный веб-фреймворк для Python, который позволяет быстро и легко создавать веб-приложения. В этом уроке мы рассмотрим основные концепции Flask и создадим простое веб-приложение.

Установка Flask

Для установки Flask используйте pip:

pip install flask

Создание простого приложения

Создадим простое веб-приложение, которое будет отображать "Hello, World!" на главной странице.

Пример кода

from flask import Flask

app = Flask('__main__')

@app.route('/')
def hello():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)

Запуск приложения

Сохраните код в файл, например app.py, и запустите его:

python app.py

После этого откройте браузер и перейдите по адресу http://127.0.0.1:5000/, чтобы увидеть сообщение "Hello, World!".

Добавление маршрутов

Вы можете добавлять дополнительные маршруты для обработки различных URL. Например:


@app.route('/about')
def about():
    return 'Это страница о нас.'

Шаблоны

Flask поддерживает шаблоны с использованием Jinja2. Вы можете создавать HTML-шаблоны и динамически генерировать контент.

Пример использования шаблонов

Создайте папку templates и файл index.html внутри неё:

        
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Главная страница</title>
</head>
<body>
    <h4>Привет, {{ name }}!</h4>
</body>
</html>

Теперь измените функцию hello, чтобы использовать шаблон:


from flask import render_template

@app.route('/')
def hello():
    return render_template('index.html', name='Мир')

Заключение
