Объектно-ориентированное программирование (ООП)

Объектно-ориентированное программирование (ООП) — это парадигма программирования, основанная на концепции "объектов", которые могут содержать данные и код: данные в виде полей (атрибутов), и код в виде процедур (методов).

Основные понятия ООП

  • Класс: Шаблон для создания объектов. Определяет набор атрибутов и методов, доступных для объектов этого класса.
  • Объект: Конкретный экземпляр класса. Объекты могут иметь разные значения атрибутов, но все они имеют одинаковые методы.
  • Атрибуты: Переменные, которые хранят состояние объекта.
  • Методы: Функции, определенные внутри класса, которые могут изменять состояние объекта или выполнять действия.

Шаг 1: Создание простого класса

Начнем с создания простого класса, который будет представлять собаку.


class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

Шаг 2: Добавление методов

Теперь добавим методы, которые позволят нашей собаке "говорить" и получать ее возраст.


    def bark(self):
        return f"{self.name} говорит: Гав-гав!"

    def get_age(self):
        return f"{self.name} - {self.age} лет."

Шаг 3: Создание объекта класса

Теперь, когда у нас есть класс, давайте создадим объект этого класса и вызовем его методы.


my_dog = Dog("Шарик", 5)

print(my_dog.bark())       # Вывод: Шарик говорит: Гав-гав!
print(my_dog.get_age())    # Вывод: Шарик - 5 лет.

Шаг 4: Наследование

Теперь давайте рассмотрим концепцию наследования, которая позволяет создавать новые классы на основе существующих. Создадим класс GuideDog, который наследует от Dog.


class GuideDog(Dog):
    def __init__(self, name, age, owner):
        super().__init__(name, age)
        self.owner = owner

    def bark(self):
        return f"{self.name}, проводник, говорит: Гав-гав!"

Шаг 5: Создание объекта класса-наследника

Теперь создадим объект класса GuideDog и вызовем его методы.


my_guide_dog = GuideDog("Рекс", 3, "Алексей")

print(my_guide_dog.bark())       # Вывод: Рекс, проводник, говорит: Гав-гав!
print(my_guide_dog.get_age())    # Вывод: Рекс - 3 лет.
print(f"Владелец: {my_guide_dog.owner}")  # Вывод: Владелец: Алексей
